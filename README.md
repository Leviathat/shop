# Getting started

**Prepare virtual environment**

    $ virtualenv venv 

    $ .\venv\Scripts\activate

    $ pip install -r requirements.txt

**Run application**

    $ python ./backend/manage.py migrate 

    $ python ./backend/manage.py runserver
